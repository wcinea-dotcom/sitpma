# Atlas des Plantes du Grand Sud d'Haïti

Site web documentaire sur les plantes médicinales, aromatiques et nutritionnelles des départements du Sud et de la Grand'Anse d'Haïti.

## 🌿 Fonctionnalités

- Base de données de 90+ plantes documentées
- Recherche multilingue (Français, Anglais, Créole)
- Filtres par famille botanique, système du corps, maladie
- Ressources documentaires (livres, articles, vocabulaire)
- Interface responsive et moderne
- Mode hors ligne avec localStorage.

## 🚀 Déploiement

### Sur Netlify (recommandé)

1. Créez un compte sur [Netlify](https://www.netlify.com)
2. Connectez votre repository GitHub
3. Cliquez sur "Deploy site"
4. C'est tout ! Netlify détectera automatiquement la configuration

### Sur GitHub Pages

1. Allez dans Settings > Pages de votre repository
2. Source: "Deploy from a branch"
3. Branch: "main", folder: "/ (root)"
4. Cliquez sur Save
